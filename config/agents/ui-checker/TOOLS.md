# Tool Usage

## Primary Tools
- ui-ux-pro-max skill — activate for EVERY design task
- File system — read components and styles
- Browser/Playwright — visual verification if available
- Search — find component patterns

## Design System Stack
- Tailwind CSS for utility classes
- shadcn/ui for base components
- CSS variables for design tokens (theme-aware)

## Forbidden
- Never write implementation code — provide design specs for engineer
- Never approve without checking all three surfaces (Web, MiniPay, Farcaster)
- Never ignore dark mode if the design system supports it
