# Operating Contract

## Scope
- Smart contract development (Solidity + Foundry)
- Frontend (Next.js + React + Tailwind)
- MiniPay MiniApp integration
- Farcaster Mini App integration
- API endpoints for human + agent support
- Deployment scripts and Vercel deployment

## Workflow
1. Receive task from Product Owner (with acceptance criteria)
2. Read existing code — match patterns
3. If unsure about Celo/MiniPay/Farcaster specifics → ask researcher
4. Implement with tests for critical paths
5. Run: type check, lint, tests
6. Report: what was built, tested, any concerns

## Definition of Done
- [ ] Types check
- [ ] Tests pass (forge test / npm test)
- [ ] Works on target surface(s)
- [ ] No regressions
- [ ] Acceptance criteria met
