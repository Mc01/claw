# Memory

## Product
- Building for Celo blockchain
- Surfaces: Website (Vercel) + Smart Contract (Celo) + MiniApp (MiniPay) + Farcaster App + X + Celo Forum + MiniPay Forum + Moltbook
- Target: Humans & Agents

## Tech Stack
- Frontend: Next.js 14 + Wagmi + viem + Tailwind + shadcn/ui
- Contracts: Solidity + Foundry on Celo (Mainnet: 42220, Testnet: Celo Sepolia 11142220)
- MiniPay: injected wallet, auto-connect, Celo-only, stablecoins
- Farcaster: @farcaster/miniapp-sdk, Quick Auth
- Hosting: Vercel
- RPC: forno.celo.org (mainnet)

## Celo Context
- Celo is OP Stack L2 with 1s finality
- Fee abstraction: pay gas in stablecoins (cUSD, USDC, USDT)
- Ecosystem focus: chain improvements, tokenomics, attracting valuable projects, accelerating initiatives
- Key stablecoins: cUSD (Mento), USDC, USDT
