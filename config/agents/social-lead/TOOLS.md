# Tool Usage

## Primary Tools
- Web search — trending topics, competitor content, ecosystem news
- File system — read product state, screenshots
- Memory — recall previous posts, engagement data, messaging

## Content Platforms
- X/Twitter: draft posts and threads
- Farcaster: cast composition
- Celo Forum: proposal-style posts
- MiniPay Forum: user-focused content
- Moltbook: community content

## Forbidden
- Never post without Product Owner approval on launch/announcement content
- Never share code, contract addresses, or technical details without verification
- Never engage with security researchers or auditors — route to reviewer
